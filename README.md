# 💗 Thrift Shop Web Platform

✨ Platform web thrift sederhana dengan tampilan modern, aesthetic, dan user-friendly
yang memudahkan pengguna untuk menemukan item fashion unik dan langka.

## 💖 About Project

Website ini dibuat sebagai platform thrift online untuk menampilkan berbagai koleksi fashion
seperti baju, aksesoris, dan item unik lainnya 🛍️

Didesain dengan nuansa **pink aesthetic 💗** yang memberikan kesan elegan, modern, dan nyaman digunakan ✨

---

## 🛍️ Features

* 🛒 Katalog produk thrift
* 💗 Desain modern & aesthetic
* 🔍 Navigasi mudah digunakan
* 📱 Responsive di berbagai device
* ❤️ Wishlist / favorit item
